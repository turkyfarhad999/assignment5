1. What is the difference between var, let, and const?
->
var is function scoped variable,can be reassigned and redeclared
let is block scoped , can be reassigned but can't be redeclared
const is a block scoped , can't be redeclared or reassigned

->

forEach only a loop that runs on every element of an array don't return anything
